## Deep learning com pytorch

Esses estudos foram realizados com o acompanhamento dos instrutores da alura.

Inclui treinamento de modelos de redes neurais a partir de um perceptron simples até camadas mais complexas na rede.
O estudo inclui modelos de regressão e classificação além de motores para otimizar os modelos de aprendizagem.

Por fim foi possível criar um modelo capaz de prever a quantidade de bicicletas que serão alugadas por semana dado uma determinada condição climática de uma região.
